# Sidebar Baseline Checklist

Date: 2026-03-17

This baseline is used to confirm visual polish changes do not alter sidebar behavior.

- Launch selection defaults to `All Photos`.
- Sidebar sections remain expanded after refresh/update.
- Sidebar selection drives main content routing for:
  - `All Photos`
  - `Recents`
  - `Favourites`
  - `Screenshots`
  - `Log`
- `All Photos` badge updates with indexed asset count changes.
- Existing selection remains selected when index count updates.
