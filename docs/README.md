# Docs Index

Last reviewed: 2026-03-24

This folder now contains only active operational/reference docs.

## Canonical Docs

- `../SPEC.MD` — product and engineering specification.
- `../CLAUDE.md` — implementation guardrails and coding rules.
- `../ROADMAP.md` — single planning source.
- `CURRENT_STATE.md` — explicit snapshot of what is implemented today.
- `ARCHITECTURE.md` — runtime architecture and module boundaries.
- `DEPENDENCY_POLICY.md` — SharedUI dependency mode and workflow.
- `RELEASE_CHECKLIST.md` — release runbook.
- `RELEASE.md` — signing/notarization packaging details.
- `Engineering Baseline.md` — baseline engineering constraints.
- `Swift6 Migration Backlog.md` — Swift 6 strictness follow-ups.

## Archive Policy

- Superseded design notes, handoff docs, and one-off investigations are moved to:
  - `docs/_archive/<YYYY-MM-DD>-<topic>/`
- Archived docs are historical only and are not planning sources.
- If a document contradicts `SPEC.MD`, `CURRENT_STATE.md`, or `ROADMAP.md`, treat the archived document as obsolete.
